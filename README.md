### Despliegue local

Clonar el proyecto

```bash
  git clone https://github.com/Natalonsop/alternovaShop.git
```

Instalar Json server

```bash
  npm instal -g json-server
```

Instalar dependencias

```bash
  npm install
```
Iniciar servidor local 

```bash
  json-server --watch src/assets/data/db.json
```

Iniciar proyecto

```bash
  npm run dev
```
